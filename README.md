## 🧱 Field summary

- `domain`, Fully qualified domain name.
- `status`, Current acquisition state for the domain in the public extract.
- `purchase_price`, Visible purchase price when available.
- `renewal_price`, Visible renewal price when available.
- `attractiveness`, Public composite naming band used as a decision-support signal.
- `demand`, Public buyer-pressure band when available.
- `length`, Character count without the TLD.
- `registrar`, Registrar name when known.
- `created_at`, Creation timestamp when known.
- `expires_at`, Expiry timestamp when known.

## ⚠️ Methodology and caveats

This is a focused set of one-word .science domain names, spanning short single words and compact compounds such as dogsit.science, geton.science, and Trex.science. With 12,220 names and a median asking price near $632, the selection sits in an accessible range for early-stage buyers comparing pricing and renewal before committing. For investors, the .science extension offers broad name availability at low entry cost, useful for building a small position across several names. For founders, many entries are short, easy to spell, and ownable now, making them practical shortlist candidates for niche science, research, or education-adjacent projects. Because .science is a non-mainstream extension, weigh brand trust and long-term renewal cost alongside price when comparing names.
